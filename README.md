Задание: Реализация класса на языке C++ для управления связанным 
списком блоков управления процессами (PCB). PCB (Process Control Block) — это структура данных, 
используемая операционной системой для хранения информации о состоянии процесса. Эта 
информация включает идентификатор процесса, его статус, счетчик команд, регистры CPU и 
другие данные, необходимые для управления выполнением процесса. 

Функционал: 
Программа позволяет управлять списком процессов через консольное меню.
Можно добавлять новые процессы с уникальным ID, именем, статусом и состоянием регистров. Процессы автоматически сортируются по ID.
Можно удалять процессы по их ID и просматривать весь список. 
Программа обрабатывает ошибки: не даёт добавить процесс с существующим ID, сообщает если процесс для удаления не найден, корректно реагирует на неправильный ввод чисел.
Техническая часть:
Реализован на C++ с использованием связных списков.
Состоит из трёх основных классов: PCB (хранит данные процесса), ListNode (узел списка с данными и указателем на следующий), ProcessList (управляет списком).
Используются указатели для связи узлов, динамическое выделение памяти new/delete.
Вся память очищается в деструкторе. Файлы: PCB.h, ListNode.h/cpp, ProcessList.h/cpp, main.cpp
